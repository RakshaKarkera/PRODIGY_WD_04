Stop Watch

The stopwatch project is a web-based application built using HTML, CSS, and JavaScript that allows users to start, pause, reset, and record lap times. Tic-Tac-Toe Game

The Tic-Tac-Toe project is a web-based game created using HTML, CSS, and JavaScript, offering an interactive and visually appealing experience. Players can take turns placing "X" and "O" on a 3x3 grid, with the game dynamically checking for win conditions and declaring the result. The interface includes a play-again button for easy game resets and a stylish design to enhance user engagement.

Weather App

The Weather App project is a web application that provides real-time weather information based on user-inputted locations. Using the OpenWeatherMap API, it fetches and displays data such as temperature, humidity, and wind speed, along with an appropriate weather icon. The user interface is designed to be clean and responsive, with a smooth fade-in animation for displaying the weather information.

Fresh Menu Website

The Fresh Menu project is a comprehensive website featuring sections like Home, Menu, Trends, Reviews, Blogs, and a Footer. The HTML and CSS code structures are well-organized to create a visually appealing and functional webpage, ensuring a smooth user experience.
